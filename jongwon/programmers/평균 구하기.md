# [프로그래머스] 평균 구하기

</br>

### 문제 설명
정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.

### 제한사항
arr은 길이 1 이상, 100 이하인 배열입니다.
arr의 원소는 -10,000 이상 10,000 이하인 정수입니다.

</br>

### 입출력 
| b | return |
|:---:|:---:|
| [1,2,3,4] | 2.5 |  
| [5,5] | 5 | 

<br>


### 풀이

```python
def solution(arr):
    answer = 0
    
    for i in arr:
        answer += i
    result = answer / len(arr)
    
    return result
```

</br>

### 다른 사람 풀이 분석

```python

def solution(arr):
    return sum(arr) / len(arr)

```
