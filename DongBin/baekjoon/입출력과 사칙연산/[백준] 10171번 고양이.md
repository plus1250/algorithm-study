# [백준] 10171번 \, ' 등의 문자에 주의하며 고양이를 출력하는 문제

## ● 출처
[[백준] 10171번 고양이](https://www.acmicpc.net/problem/10171)  

## ● 문제

## ● 입력
없음

## ● 출력
고양이를 출력한다.

## ● 예제 입력

```
(  /  )
 \(__)|
```

